BugNet

BugNet is a bug tracking software built with ASP.NET, MVC, C#, and SQL. It features a role-based security system that restricts user access based on permission-based roles, including Admin, Project Manager, Developer, and Submitter. Users can submit tickets that are tracked from Open to Complete status.

The system was developed in 3 weeks using C#, .Net, HTML, Javascript, Bootstrap, MVC, and SQL Server. The database is hosted on Azure. Please visit the site.
